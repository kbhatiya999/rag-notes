# 1. Introduction

LightRAG runs a personal AI search server on your Mac. These notes cover how to install it, configure it with an AI model, and wrap it into a double-clickable Mac app using a tool called Platypus.

## Prerequisites

1. **AI Provider**: You need an API key (OpenAI, Google Gemini) or a local provider installed (Ollama).
2. **Platypus**: Download [Platypus](https://sveinbjorn.org/platypus). Open the `.zip` and drag `Platypus.app` into your Applications folder using Finder.
